# `codyai setup`

Initialize `~/.codyai/codyai.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
codyai setup
codyai setup --workspace ~/.codyai/workspace
```

To run the wizard via setup:

```bash
codyai setup --wizard
```
